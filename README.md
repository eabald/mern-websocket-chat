# MERN-WEBSOCKET-CHAT

## Development

### Perquisites

* Docker installed.
* Node and npm installed.

### Start in development mode

Run:

```bash
 ./start_db.sh
```

Install all dependencies:

```bash
cd backend && npm i
```

and

```bash
cd client && npm i
```

In separate terminals start development servers:

```bash
cd backend && npm start
```

and

```bash
cd client && npm start
```
